Assumptions

This is designed to handle the spreadsheet (.xlsx) supplied by Home Affairs.

  • Electorate names that can't be found (e.g. spelling errors) are currently being logged to SEQ and will not be uploaded. This will not prevent the rest of the valid electorates from being uploaded.
  • Each worksheet will create a report.
  • It makes some assumptions about worksheet names and which reports they map to. Document those here.
